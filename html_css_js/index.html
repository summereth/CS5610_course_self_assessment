Positioning Example

Static Positioning

This is a statically positioned element (default behavior).

When to use: Default positioning for all elements. Best for normal document flow where elements follow each other naturally.

Relative Positioning

This element is positioned relative to its normal position.

When to use: When you need to make small adjustments to an element's position without affecting other elements, or when creating a positioning context for absolute children.

Absolute Positioning

This element is positioned relative to its closest positioned ancestor.

When to use: For elements that need to be precisely positioned relative to a parent container, like modals, tooltips, or decorative elements.

Fixed Positioning

This notification stays fixed on screen while scrolling.

When to use fixed positioning: For elements that should remain visible while scrolling, such as navigation bars, chat widgets, or important notifications.
